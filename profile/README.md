## Aps Kullanım Örnekleri

Yükleme
```bash
sudo aps install steam
```
Kaldırma
```bash
sudo aps remove steam
```
Güncelleme
```bash
sudo aps up steam
```
Güncellemeyi Silme
```bash
sudo aps rm-up steam
```
Yardım Alma
```bash
aps
```
```bash
aps help
```
